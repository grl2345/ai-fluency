# AI Fluency Assessment

基于六维能力模型的 AI 流利度测评 Web 应用，帮助用户了解自身 AI 素养水平并获得学习建议。

## 功能

- 落地页介绍测评维度与等级体系
- 交互式测评流程
- 结果页展示各维度得分与等级解读

## 技术栈

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui

## 本地开发

```bash
pnpm install
pnpm dev
```

浏览器访问 [http://localhost:3000](http://localhost:3000)。

## 构建与部署

```bash
pnpm build
pnpm start
```
